# M_AudioMaterialMeter 美术文档

## 📋 材质简介

`M_AudioMaterialMeter` 是用于音频电平表显示的 UI 材质，常见于游戏中的音量设置界面、音频可视化面板等场景。

**材质路径**: `/AudioWidgets/AudioMaterialSlate/M_AudioMaterialMeter`

---

## 🎨 视觉效果

```
┌─────────────────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░░░░░░░░░░░░░░░ │  ← 电平表示例
│  ↑                                  │
│  蓝色 → 橙色 → 红色                  │
│  低      中      高                  │
└─────────────────────────────────────┘
```

**颜色渐变**:
- **低电平** (0-33%): 蓝色 - 安全区域
- **中电平** (33-66%): 橙色 - 警告区域  
- **高电平** (66-100%): 红色 - 危险/峰值区域

---

## 🎛️ 参数调节指南

### 颜色定制

| 参数 | 效果 | 建议值范围 |
|------|------|------------|
| `A (V3)` | 低电平颜色 | 冷色调 (蓝、绿、青) |
| `B (V3)` | 中电平颜色 | 暖色调 (黄、橙) |
| `C (V3)` | 高电平颜色 | 警告色 (红、白) |
| `DotsOffColor` | 未激活区域颜色 | 通常为黑色或透明 |

### 数值控制

| 参数 | 效果 | 调节建议 |
|------|------|----------|
| `VALUE` | 当前电平 | 由程序动态控制 |
| `NumLinesX` | 分段数量 | 5-20，数值越大越平滑 |
| `GlowAmt` | 发光强度 | 0-5，过高会过曝 |
| `GradientPow` | 渐变过渡 | 1-5，影响颜色过渡锐度 |

## ⚠️ 美术注意事项

### 颜色选择
1. **对比度**: 确保三个颜色区域有明显区分
2. **HDR考虑**: 高电平颜色可设置超过1.0的值产生发光效果
3. **色盲友好**: 考虑红绿色盲用户，可添加图案或亮度区分

### 发光效果
1. `GlowAmt` 过高会导致 UI 过曝
2. 配合 Bloom 后期效果使用效果更佳
3. 移动端建议降低或关闭发光

### 性能考虑
1. 该材质用于 Slate UI，对性能影响较小
2. 大量同时使用时注意 Draw Call

## 🔧 程序接口

材质实例动态控制示例 (伪代码):

```cpp
// 设置电平值 (0.0 - 1.0)
MaterialInstance->SetScalarParameterValue("VALUE", AudioLevel);

// 设置分段数
MaterialInstance->SetScalarParameterValue("NumLinesX", 10);

// 动态改变颜色
MaterialInstance->SetVectorParameterValue("A (V3)", FLinearColor::Blue);
```
