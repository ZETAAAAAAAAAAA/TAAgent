# M_Water 美术文档

## 效果预览

这是一个用于水面渲染的材质，支持动态流向、深度着色和菲涅尔反射效果。

---

## 可调参数

### 🎨 WaterColor (水体颜色)

| 属性 | 值 |
|------|-----|
| 类型 | 颜色 |
| 默认值 | 深青色 (RGB: 6, 32, 45) |
| 效果 | 控制水体的基础颜色 |

**调节建议**:
- 清澈湖水：增加绿色分量
- 深海：降低整体亮度，增加蓝色
- 污染水域：增加绿色/黄色

---

### 📏 MaxDepth (cm) (最大深度)

| 属性 | 值 |
|------|-----|
| 类型 | 数值 |
| 默认值 | 200 cm |
| 范围 | 50 ~ 1000+ |
| 效果 | 控制水深影响范围 |

**视觉表现**:
- **小值 (50-100)**: 浅水效果，快速过渡到底色
- **中值 (200-400)**: 自然水深过渡
- **大值 (500+)**: 深水效果，渐进变化

---

### 💧 Turbidity (浊度)

| 属性 | 值 |
|------|-----|
| 类型 | 数值 |
| 默认值 | 1.0 |
| 范围 | 0 ~ 5+ |
| 效果 | 控制水体浑浊程度 |

**视觉表现**:
- **0-0.5**: 清澈水体，高透明度
- **1.0**: 正常水体，适度浑浊
- **2-3**: 浑浊水体，降低可见度
- **5+**: 非常浑浊，几乎不透明

---

### 🌀 FlowDirection (流向角度)

| 属性 | 值 |
|------|-----|
| 类型 | 数值（度） |
| 默认值 | 0 |
| 范围 | 0 ~ 360 |
| 效果 | 控制水面流动方向 |

**方向对应**:
- 0° / 360°: 向右流动
- 90°: 向上流动
- 180°: 向左流动
- 270°: 向下流动

---

### ⚡ FlowSpeed (流动速度)

| 属性 | 值 |
|------|-----|
| 类型 | 数值 |
| 默认值 | 0.1 |
| 范围 | 0 ~ 1+ |
| 效果 | 控制水面流动速度 |

**调节建议**:
- **静止水面**: 0
- **缓慢流动**: 0.05 - 0.15
- **正常流速**: 0.2 - 0.5
- **湍急水流**: 0.5+

---

## 视觉效果说明

### 深度效果

材质会根据水深自动调整颜色深浅：
- **浅水区域**: 更透明，可见底部
- **深水区域**: 更浑浊，颜色更深

### 边缘反射

使用菲涅尔效果实现：
- **正视水面**: 透过水面看到水下
- **斜视水面**: 反射天空/环境

### 焦散效果

自动计算光线折射产生的焦散图案。

## 使用场景

| 场景 | WaterColor | MaxDepth | Turbidity | FlowSpeed |
|------|------------|----------|-----------|-----------|
| 清澈泳池 | 浅蓝 | 150 | 0.2 | 0.05 |
| 山间溪流 | 青绿 | 100 | 0.5 | 0.3 |
| 平静湖泊 | 深青 | 300 | 1.0 | 0.02 |
| 浑浊河流 | 黄绿 | 200 | 2.5 | 0.15 |
| 深海 | 深蓝黑 | 500 | 0.8 | 0.01 |

---

## 美术调节工作流

### 1. 确定基础色调
```
WaterColor → 选择符合场景氛围的颜色
```

### 2. 设置深度效果
```
MaxDepth → 根据实际水深设置
Turbidity → 调整水体质感
```

### 3. 添加动态效果
```
FlowDirection → 确定流向
FlowSpeed → 设置流速
```

### 4. 精细调整
```
微调各参数 → 达到理想效果
```

---

## 注意事项

⚠️ **性能提示**:
- 这是一个半透明材质，存在 overdraw 开销
- 建议控制水面网格的复杂度

⚠️ **视觉提示**:
- 确保场景中有足够的环境光
- 水面下方物体需要正确设置深度
